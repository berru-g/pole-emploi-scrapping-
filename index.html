Tool

Scrapping des offres d'emploi Linkedin ciblé par des mots clefs et réponse automatique

Outil simple à mettre en place avec python et API linkedin

Etapes:

1 - Inscription Linkedin For developer.

2 - Install lib

    pip install bs4
    pip install requests
    pip install linkedin

3 - Définire les mots-clés à rechercher dans les annonces d'emploi

4 - Recherchez les annonces d'emploi avec les mots-clés spécifiés

5 - Imprimez les résultats

6 - Envoyer une réponse automatique avec votre site Web

7 - Récupérer toutes les conversations liées à votre compte LinkedIn

8 - Filtrer les conversations avec les recruteurs

9 - Extraire les liens des annonces d'emploi correspondantes
